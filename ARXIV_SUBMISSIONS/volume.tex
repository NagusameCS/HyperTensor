\section{Paper I: Calibration-Free Low-Rank Attention Compression}
\subsection{Motivation and Method}
Transformer inference is memory-bandwidth bound during autoregressive decode. Each token requires loading full attention weight matrices $W_Q, W_K, W_V \in \mathbb{R}^{d \times d}$ from GPU memory. For Llama-3.1-8B with $d = 4096$, this is $3 \times 4096^2 \times 2$ bytes $\approx 100$ MB per attention layer. Our method projects attention weights onto their own principal components with no calibration data. For each weight matrix $W$, we compute $G = W W^T$, perform eigendecomposition $G = U \Lambda U^T$, and retain the top-$k$ eigenvectors: $B = U_{[:,:k]}$. The effective weight becomes $W_k = B B^T W$, storing only $B$. The optimal rank $k^*$ is predicted analytically: $k^* = \mathrm{L2\_MB} \times 42.7$.

\section{Paper XIII: Orthogonal Geodesic Deviation (Safe OGD)}
\textit{v1.0, 100\% complete.}

\subsection{Method}
$P_{\mathrm{safe}} = I - Q_f Q_f^T$ where $Q_f$ = orthonormal basis for forbidden subspace. Safe projection: $h_{\mathrm{safe}} = P_{\mathrm{safe}} h$.

\subsection{Theorem (Safety)}
For any $h$, any $v$: $h_{\mathrm{safe}} = P_{\mathrm{safe}}(h + \alpha v)$ has zero TEH for all $\alpha$.
\textit{Proof:} $Q_f^T P_{\mathrm{safe}} = Q_f^T(I - Q_f Q_f^T) = Q_f^T - Q_f^T = 0$. $\square$
